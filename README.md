# 🇺🇸 English Documentation

## ✨ Features

- Fully Offline — No Internet Required
- Drag & Drop Folder Support
- Generate Text-Based Directory Tree Automatically
- Copy Output to Clipboard
- Export Tree as TXT File
- Modern Responsive UI
- Lightweight & Fast
- No Backend Required
- Supports Nested Directories
- Works in Modern Browsers

---

## 📦 Requirements

No installation required.

You only need:

- Modern Web Browser
  - Google Chrome
  - Microsoft Edge
  - Brave
  - Opera

> Safari and Firefox may have limited folder drag-and-drop support.

---

## 🚀 Usage

### 1. Download or Create the Project

Save the file as:

```text
index.html
```

### 2. Open in Browser

Double-click the file or open it using your browser.

---

### 3. Drag & Drop a Folder

Drag any folder into the application.

The tool will automatically generate a text-based directory tree.

---

## 📋 Example Output

```text
project/
```
